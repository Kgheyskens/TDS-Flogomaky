## Quickstart

1. Clone the repo
   git clone <repo-url>
2. Add code for your stack:
   - For Node: create package.json, implement entry in `src/` or `lib/`
   - For Python: create a virtualenv and `src/` or package layout
3. Install dependencies (example for Node)
   npm install
4. Run locally (example)
   npm start

## What to do next

- Add your application code under a `src/` folder (create it if missing).
- Add a package manifest (package.json, pyproject.toml, etc.) for your chosen stack.
- Add tests and CI as needed.
